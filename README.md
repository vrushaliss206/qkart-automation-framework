# QKart – E-commerce Test Automation Framework

# Overview
This project is a test automation framework developed for an e-commerce web application similar to QKart. 
It demonstrates end-to-end UI and API testing using Selenium WebDriver, Java, and TestNG.

# Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Gradle
- Apache POI (Data-driven testing)
- Jackson (JSON processing)
- Postman (API testing)
- Git & Jenkins

# Framework Design
- Page Object Model (POM)
- Modular and reusable components
- Data-driven testing using Excel and JSON
- Cross-browser test execution support

# Test Coverage
- User Registration & Login
- Product Search
- Add to Cart
- Checkout Flow
- Order Placement
- API validation for product catalog and orders

## CI/CD
- Integrated with Jenkins for automated execution of test suites
